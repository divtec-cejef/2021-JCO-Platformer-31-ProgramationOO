\doxysubsection{Detailed Description}
Gestionnaire de déplacement manuel. 

Gestionnaire de déplacement qui se base sur le tick.

Déplace le sprite de gauche à droite jusqu\textquotesingle{}à rencontrer un obstacle. La direction est alors changée et la marche se poursuite.

La vitesse de la marche est ajustable avec \mbox{\hyperlink{class_manual_walking_handler_ad076f22e2bb712daf4e5057214b09eee}{set\+Walking\+Speed()}}.
