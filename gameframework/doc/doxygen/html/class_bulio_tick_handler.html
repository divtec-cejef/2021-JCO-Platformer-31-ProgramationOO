Description détaillée

Cette classe est permet de crée un ennemie se déplaçant de droite à gauche.

du joueur

Le Bulio se déplace toujours à droite si il ne rentre pas en collision avec un autre sprite, si telle est le cas le bulio change de dirction et continue à se déplacé dans le sens opposé du sprite touché. Il est soumis à la gravité si il n'y a pas de sol sous ses pieds il sera attiré en bas de l'écran. //!
