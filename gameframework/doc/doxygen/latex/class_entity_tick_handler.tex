\doxysubsection{Description détaillée}
Classe abstraite qui représente un gestionnaire de tick pour une entité. 

Cette classe a pour but d\textquotesingle{}être spécialisée afin de donner à un sprite un comportement à effectuer lors de chaque tick émis par la cadence.

Lorsqu\textquotesingle{}un gestionnaire est affecté à une entité (\mbox{\hyperlink{class_sprite_a824c82dee03274adfee1287bda73ff66}{Entity\+::set\+Tick\+Handler}}), la fonction init() du gestionnaire est appelée une unique fois, puis, lorsque la cadence démarre, la fonction tick() du gestionnaire est automatiquement appelée.

Depuis le gestionnaire, il est possible d\textquotesingle{}accéder au \mbox{\hyperlink{class_sprite}{Sprite}} en question avec l\textquotesingle{}attribut m\+\_\+p\+Parent\+Entity.
