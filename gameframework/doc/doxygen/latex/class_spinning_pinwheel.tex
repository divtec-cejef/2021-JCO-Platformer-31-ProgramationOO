\doxysubsection{Detailed Description}
Démontre le suivi de la souris. 

Cette classe spécialise \mbox{\hyperlink{class_sprite}{Sprite}} pour gérer le déplacement d\textquotesingle{}une roue tournante, qui suit en permanence le pointeur de la souris.

Cette classe démontre également les fonctionnalités suivantes \+:
\begin{DoxyItemize}
\item Détecter les changements d\textquotesingle{}état d\textquotesingle{}un item (\mbox{\hyperlink{class_spinning_pinwheel_a9e4ce9d580b75cf1d437fdfe4548b2ca}{item\+Change()}}, mécanisme hérité de \mbox{\hyperlink{class_sprite}{Sprite}} qui lui-\/même l\textquotesingle{}hérite de QGraphics\+Item) et plus particulièrement dans ce cas un changement de visibilité pour stopper l\textquotesingle{}animation de la roue si celle-\/ci est invisible.
\item Générer une animation à partir d\textquotesingle{}une spritesheet (dans configure\+Animation()). 
\end{DoxyItemize}
