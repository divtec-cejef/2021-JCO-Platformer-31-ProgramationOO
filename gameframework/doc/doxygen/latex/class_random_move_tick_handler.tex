\doxysubsection{Detailed Description}
Classe qui gère un déplacement aléatoire d\textquotesingle{}un sprite. 

Le sprite est déplacer tout en cherchant à éviter les collisions.

Si la destruction sur collision est activée (set\+Destroy\+On\+Collision\+Enabled), en cas de collision le sprite est détruit.
