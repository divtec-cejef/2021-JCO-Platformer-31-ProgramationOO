Description détaillée

Gestionnaire de déplacement manuel.

Gestionnaire de déplacement qui se base sur le tick.

Déplace le sprite de gauche à droite jusqu'à rencontrer un obstacle. La direction est alors changée et la marche se poursuite.

La vitesse de la marche est ajustable avec setWalkingSpeed().
