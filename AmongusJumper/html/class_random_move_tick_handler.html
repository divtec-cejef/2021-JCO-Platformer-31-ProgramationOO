Description détaillée

Classe qui gère un déplacement aléatoire d'un sprite.

Le sprite est déplacer tout en cherchant à éviter les collisions.

Si la destruction sur collision est activée (setDestroyOnCollisionEnabled), en cas de collision le sprite est détruit.
